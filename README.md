# storeapi

# Postman collection: https://www.getpostman.com/collections/16c129da1dcdf2b383aa

1. After you cloned the project in console:

$ npm install

2. Start mongo in console:

$ mongod

or 

$ mongo 
$ use storeapi

3. Start node app in console:

$ node index.js
